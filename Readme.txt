Juego Tic Tac Toe

Se basa en recrear el juego de Tic Tac Toe bajo las lineas de comando de Python, 
se realiza de manera sencilla y rápida.

Se procede a crear dos tableros, uno numerico y otro(el cual ve el ususario),
con el tablero numerico se verifica si el juego termina en empate o gana unos de
los participantes.
